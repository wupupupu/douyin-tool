# CSV 数据格式说明

## 正确格式

CSV 文件第一行必须是以下列名（顺序不限，名称必须一致）：

```
视频标题,IP角色,风格标签,播放量,点赞数,评论数,分享数,发布日期
```

## 完整示例

```csv
视频标题,IP角色,风格标签,播放量,点赞数,评论数,分享数,发布日期
甘雨古风舞蹈翻跳,甘雨,古风,5000,1200,15,50,2024-01-15
原神热舞挑战,原神,热舞,3000,800,12,30,2024-01-16
崩坏星穹铁道舞蹈,崩坏,二次元,2500,600,8,20,2024-01-17
```

## 列说明

| 列名 | 必填 | 类型 | 说明 |
|------|------|------|------|
| 视频标题 | 是 | 文本 | 视频名称 |
| IP角色 | 否 | 文本 | 用于 IP 分类统计 |
| 风格标签 | 否 | 文本 | 用于风格趋势分析 |
| 播放量 | 是 | 数字 | 视频播放次数 |
| 点赞数 | 是 | 数字 | 点赞次数 |
| 评论数 | 是 | 数字 | 评论次数 |
| 分享数 | 否 | 数字 | 分享次数 |
| 发布日期 | 是 | 日期 | YYYY-MM-DD 格式 |

## 数字格式要求

**正确：**
```
1000
2500
9999
```

**错误：**
```
1,000     （不能有逗号——系统会自动处理，但建议去掉）
1k        （不能用单位）
~1000     （不能有符号）
```

## 日期格式

必须是 `YYYY-MM-DD` 格式：
- 正确：`2024-01-15`
- 错误：`2024/1/15`、`2024年1月15日`

## 从抖音导出

1. 打开 `https://creator.douyin.com`
2. 进入「数据」→「视频数据」
3. 选择时间范围
4. 导出为 CSV
5. 如导出为 Excel 格式，请用 Excel 另存为 CSV

## 常见错误

**列名不匹配：** 列名必须完全一致（中文标点）。系统会忽略多余的列。

**编码问题：** 如上传后中文乱码，请用记事本打开 CSV 文件，另存为 UTF-8 编码。

**空行问题：** 删除文件末尾的空行和空列。
