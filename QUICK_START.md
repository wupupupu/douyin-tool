# 5 分钟快速上手

## 步骤 1：启动应用

```bash
pip install -r requirements.txt
python app.py
```

浏览器打开 `http://localhost:5000`

## 步骤 2：导出抖音数据

1. 打开抖音创作者中心 `https://creator.douyin.com`
2. 进入「数据」→「视频数据」
3. 选择时间范围，导出 CSV

## 步骤 3：上传分析

1. 点击「数据分析」标签
2. 点击上传区域（或拖拽 CSV 文件）
3. 等待分析完成，自动展示统计图表

## 步骤 4：使用选题库

1. 点击「选题库」标签
2. 填写标题、选择分类
3. 可选添加参考链接
4. 点击「添加选题」

## 步骤 5：规划发布

1. 点击「发布日历」标签
2. 选择日期、输入标题
3. 添加备注（道具、场景等）
4. 发布后点击「标记已发布」

## 在 iPad / iPhone 上使用

如果部署到 Railway 后：
1. Safari 打开应用链接
2. 点击分享按钮 →「添加到主屏幕」
3. 像 App 一样直接打开

## 常见问题

**Q: 上传 CSV 失败？**
检查 CSV 列名是否为：视频标题,IP角色,风格标签,播放量,点赞数,评论数,分享数,发布日期

**Q: 图表不显示？**
刷新页面后重新上传。确保数据至少有 3 条记录。

**Q: 数据会丢失吗？**
数据保存在 `data_storage.json` 文件中，关闭服务器不会丢失。建议定期点击「导出所有数据」备份。
